# Juntos Hacia Dios

Sitio web cristiano y católico para compartir cantos, letras, acordes, artistas, categorías, álbumes y recursos de alabanza.

## Acerca del proyecto

**Juntos Hacia Dios** nace con el propósito de ayudar a músicos, ministerios, coros y comunidades de fe a encontrar cantos de una forma sencilla, ordenada y accesible.

El sitio permite explorar canciones por artista, categoría o tipo de canto, además de consultar letras, acordes, tono original, capo y enlaces de apoyo como tutoriales o videos.

## Funciones principales

- Cancionero con letras y acordes.
- Búsqueda de canciones.
- Filtros por tipo de canto.
- Páginas de artistas y ministerios.
- Categorías de cantos.
- Álbumes o carpetas por artista.
- Links de tutoriales y recursos.
- Transposición de acordes.
- Soporte para capo.
- Panel de administración privado.
- Integración con Supabase.

## Tecnologías

- HTML
- CSS
- JavaScript
- Supabase
- GitHub Pages

## Objetivo

Facilitar el acceso a cantos de oración, adoración y alabanza para acompañar momentos personales, grupos de oración, ministerios de música y celebraciones.
